Pokemon REST-API

Simple API for data fetching from MongoDB

Examples for HTTP-methods:

GET: one listed item: /api/get/<add id>

Example:

/api/get/5e9c425c2bb02b0a242ea8e8 < ObjectId from the MongoDB

Result in HTML-page/Console:

{ "type": [ "Poison", "Grass" ], "_id": "5e9c425c2bb02b0a242ea8e8", "name": "Bulbasaur", "jname": "フシギダネ", "weight": 6, "pokedex": 1 }

GET: all listings: /api/getAll

POST: add new listing to DB: /api/add

PUT: update listed item in the DB: /api/update/<add id>

DELETE: listing form the DB: /api/delete/<add id>
